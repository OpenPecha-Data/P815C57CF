བསྡུ་སྒྲིག་གསལ་བཤད།

བསྟན་བཅོས་འདིའི་ཁུངས་ནི། འབུམ་སྡེ་གོ་དགུའི་མངའ་བདག་ཀུན་མཁྱེན་དབྱིག་གཉེན་གྱི་ཐུགས་སྲས་ཚད་མ་རང་ལས་མཁས་པ་སློབ་དཔོན་ཕྱོགས་གླང་གིས་མཛད་པའི་ཚད་མ་མདོ་ཡིན་ལ། དེའི་དོན་རྒྱས་པར་མཛད་པའི་ཤིང་རྟའི་ལམ་སྲོལ་ནི་རྒྱ་བོད་མཁས་པའི་གཙུག་རྒྱན་དཔལ་ཆོས་ཀྱི་གྲགས་པའི་ཚད་མ་སྡེ་བདུན་ཡིན། སྤྱིར་ཚད་མའི་བསྟན་བཅོས་ཀྱི་ལུས་ཡོངས་རྫོགས་མངོན་སུམ་དང༌། རང་དོན་རྗེས་དཔགགཞན་དོན་རྗེས་དཔག་གི་ལེའུ་སྟེ་དེ་གསུམ་དུ་འདུས་པས་ན། བཀྲལ་བྱ་ཚད་མ་རྣམ་པར་ངེས་པ་ནི་ཚད་མའི་ལུས་ཡོངས་སུ་རྫོགས་པའི་བསྟན་བཅོས་རྨད་དུ་བྱུང་བ་ཞིག་ཡིན། དེ་ལའང་སྔ་འགྲེལ་རྒྱ་གར་མཁས་པའི་བསྟན་བཅོས་ཇི་སྙེད་ཡོད་པ་རྣམས་ཀྱི་རྩ་འགྲེལ་ཤིན་ཏུ་བརླིང་བའི་གོ་བར་དཀའ་བ་ཙམ་དུ་མ་ཟད་འགྲེལ་བཤད་དག་ཏུ་མ་ཐོན་པའི་ལེགས་པར་བཤད་པ་མང་པོ་དག་ཇི་ལྟར་ཡོད་པའི་ཚུལ་ནི། ལོ་ཙཱ་བ་ཆེན་པོས། ཇི་སྐད་དུ། གཞུང་རྩོམ་རྣམས་ཀྱི་བློ་ལ་ཡང༌། ། ཇི་བཞིན་ལེགས་པར་མི་གསལ་བ། ། ཚད་མའི་དེ་ཉིད་དཀའ་བ་རྣམས། ། ཇི་བཞིན་དུ་ནི་འདིར་ངེས་བྱ། ། ཞེས་རྩོམ་པ་དམ་བཅས་ནས་ཚད་མའི་བསྟན་བཅོས་སོགས་གཞུང་ཕྲན་མང་པོ་མཛད། དེ་ལྟར་མཛད་པ་ནི་བོད་ཡུལ་དུ་རྟོག་གེ་ཚད་མའི་སྲོལ་འབྱེད་ཀྱི་ཐོག་མ་སྟེ་གཞན་དེ་ལས་སྔ་བ་བྱུང་མེད། བོད་དུ་རྟོག་གེ་ཚད་མའི་སྲོལ་ཐོག་མར་འབྱེད་པ་པོ་རྔོག་ལོ་ཙཱ་བ་བློ་ལྡན་ཤེས་རབ་ནི། བོད་ཀྱི་རབ་བྱུང་དང་པོའི་ས་ཕག་ ( ༡༠༥༩ ) ལོར་འཁྲུངས་ཤིང༌། རབ་བྱུང་གཉིས་པའི་ས་གླང་ ( ༡༡༠༩ ) ལོར་སྐུ་གཤེགས། རང་ཉིད་ཀྱི་ཁུ་བོ་རྔོག་ལེགས་པའི་ཤེས་རབ་ཀྱིས་ཕྱི་ལོ་ ( ༡༠༧༣ ) ལོར་ཕྱག་བཏབ་པའི་གསང་ཕུར་ཆོས་ཞུགས་མཛད། དེ་ནས་ཁུ་བོའི་ལམ་སྟོན་ལྟར་རྒྱ་གར་དང་ཁ་ཆེར་ལོ་བཅུ་བདུན་གྱི་རིང་མདོ་སྔགས་སྒྲ་ཚད་སོགས་ལ་དཀའ་སྤྱད་ཐུགས་བརྩོན་བློ་ཡུལ་དུ་མི་ཤོང་བར་གསན་བཞེས་བརྐྱངས་པས་མཁས་པའི་རྩེ་མོར་སོན། དགུང་ལོ་སོ་ལྔ་པ་

( ༡༠༩༢ ) ལོར་བོད་དུ་ཕྱིར་ཕེབས་མཛད། བཀའ་བསྟན་བཅོས་ཤོ་ལོ་ཀ་འབུམ་ལྷག་པར་བསྒྱུར་ཡོད་པའང་གྲོ་ལུང་པས་མཛད་པའི་འཇིག་རྟེན་མིག་གཅིག་བློ་ལྡན་ཤེས་རབ་ཀྱི་རྣམ་ཐར་ལས། དམ་ཆོས་བསྒྱུར་ལ་མངའ་བརྙེས་པ། ། རྒྱལ་ཡུམ་གཉིས་དང་བསྟན་བཅོས་གཞུང༌། ། འབུམ་ཕྲག་གཅིག་དང་ཁྲི་ཕྲག་གསུམ། ། སྟོང་ཕྲག་བདུན་ནི་ལྷག་བཅས་བསྒྱུར། ། ཞེས་གསུངས། ཁོང་གི་བརྩམས་ཆོས་ནི་རྣམ་ཐར་དང་དཀར་ཆག་ཁག་ཏུ་གསལ་བ་ལྟར་ན། ཚད་མའི་གཞུང་འགྲེལ་ལ། ཚད་མ་རྣམ་པར་ངེས་པའི་བསྡུས་དོན་དང་རྣམ་བཤད་ཆེན་མོ། རིགས་ཐིགས་ཀྱི་འགྲེལ་པ་དང་བཅས་པའི་བསྡུས་དོན། ལེའུ་དང་པོའི་སྟོད་ཅུང་ཟད་ཀྱི་རྣམ་བཤད། ཆོས་མཆོག་ཆེན་པོའི་ངག་དང་པོའི་ཚིགས་སུ་བཅད་པ་བདུན་གྱི་བཤད་པ་དང༌། དེའི་གཞན་སེལ་བའི་སྐབས་ཅུང་ཟད་བཤད་པ། ཆོས་མཆོག་གི་ཚད་མ་བརྟག་པ་ཆུང་ངུའི་བསྡུས་དོན། སེལ་བ་སྒྲུབ་པའི་བསྡུས་དོན་དང་སྟོད་ཀྱི་བཤད་པ་དང༌། སྐད་ཅིག་འཇིག་པ་སྒྲུབ་པའི་བསྡུས་དོན་དང༌། བྲམ་ཟེ་ཆེན་པོའི་སེལ་བ་སྒྲུབ་པའི་བསྡུས་དོན་དང་སྟོད་ཀྱི་རྣམ་བཤད་དང༌། འབྲེལ་བ་སྒྲུབ་པའི་བསྡུས་དོན་དང་སྟོད་ཀྱི་རྣམ་པར་བཤད་པ་རྣམས་སོ། ། བྱམས་ཆོས་དང་དབུ་མའི་སྐོར་ཇི་ཡོད་རྣམས་མཛད་རྣམ་མདོར་བསྡུས་སྐབས་སུ་གཟིགས། ད་ལྟ་ང་ཚོས་འཚོལ་བསྡུ་ཐུབ་པ་ནི། བོད་ལྗོངས་དཔེ་མཛོད་ཁང་དུ་བཞུགས་པ་དག་ཡིག་ཉེར་མཁོ་བསྡུས་པ་དང༌། མཁན་པོ་ཚུལ་ཁྲིམས་རྒྱལ་མཚན་མཆོག་ནས་གཡར་གནང་མཛད་པ་ཤེས་རབ་སྙིང་པོའི་འགྲེལ་པ། འབྲས་སྤུངས་གནས་བཅུ་ལྷ་ཁང་དུ་བཞུགས་པའི་མངོན་རྟོགས་རྒྱན་གྱི་དཀའ་འགྲེལ་རིན་ཆེན་སྒྲོན་མེ་དང༌། མདོ་སྡེ་རྒྱན་གྱི་དོན་བསྡུས་དང༌། དབུས་མཐའ་རྣམ་འབྱེད་ཀྱི་དོན་བསྡུས་དང༌། སྤོ་མི་སྤྱན་དྲུག་ལུང་དུ་གནས་སྡོད་བཀྲ་ཤིས་དབང་རྒྱལ་ནས་མ་དཔེ་མཁོ་འདོན་གནང་བ་རྒྱུད་བླ་མའི་འགྲེལ་པ་དང༌། རྒྱལ་རྩེ་དཔལ་འཁོར་ཆོས་སྡེར་བཞུགས་པ་ཚད་མ་རྣམ་ངེས་ཀྱི་བསྡུས་དོན་དང༌།

སེ་ར་དགོན་དུ་བཞུགས་པའི་ཚད་མ་རྣམ་ངེས་ཀྱི་འགྲེལ་ཆེན་དང༌། འབྲས་སྤུངས་གནས་བཅུ་ལྷ་ཁང་དུ་བཞུགས་པའི་སྤྲིང་ཡིག་བདུད་རྩིའི་ཐིགས་པ་བཅས་ཆོས་ཚན་དགུ་པོ་འདི་བཀའ་གདམས་དཔེ་དཀོན་ གཅེས་བཏུས་ཀྱི་དེབ་བཞི་པར་བཞག་ཡོད། སེ་ར་དགོན་ནས་གཡར་གནང་ཞུས་པའི་མ་དཔེ་འདིའི་མཇུག་ནས་ལྡེབ་གསུམ་ཙམ་ཚང་མེད་པ་དང༌། ཀྲུང་གོའི་བོད་ཀྱི་ཤེས་རིག་དཔེ་སྐྲུན་ཁང་ནས་སྤྱི་ལོ་ ( ༡༩༩༢ ) ལོར་པར་བསྐྲུན་ཞུས་པའི་དེབ་འདིའི་དབུ་ནས་ལྡེབ་གཅིག་ཚང་མེད་ནའང་དེ་གཉིས་ཕན་ཚུན་བཏུས་ནས་ཆ་ཚང་བར་བྱས་ཡོད། གོང་ཞུས་ཀྱི་མ་དཔེ་འདི་རྣམས་སྤྱི་ལོའི་དུས་རབས་བཅུ་གཉིས་པའི་ཡས་མས་སུ་བྲིས་པ་ཞིག་ཡིན་པས། སྐབས་དེར་བོད་ཡིག་འགྱུར་རྙིང་པ་ནས་གསར་པར་འཕོ་བ་དང༌། བརྡ་རྙིང་པ་ནས་གསར་པར་འགྱུར་བའི་བར་བརྒལ་དུས་སུ་ཁེལ་ཡོད་སྟབས། མ་དཔེ་འདི་རྣམས་ལ་མིང་ཚིག་མི་འདྲ་བ་དང༌། བརྡ་གསར་རྙིང་འདྲེས་མ་མང་ཙམ་མཆིས་ཀྱང་མ་དཔེ་ཇི་མ་ཇི་བཞིན་བཀའ་གདམས་གསུང་འབུམ་ཕྱོགས་སྒྲིག་ཐེང་དང་པོར་པར་བསྐྲུན་ཞུས་ཟིན་པས་འདིར་ཚིག་དོན་གཉིས་ཀར་འཁྲུལ་བ་མི་ཡོང་བའི་ཆེད་སྲོལ་རྒྱུན་གྱི་བརྡ་དག་དང་བསྟུན་ཏེ་དག་པར་བཅོས་ནས་རྒྱ་ཆེའི་གཟིགས་པ་པོའི་སྤྱན་ལམ་དུ་པར་སྐྲུན་ཞུས་པ་ཡིན། དཔལ་བརྩེགས་བོད་ཡིག་དཔེ་རྙིང་ཞིབ་འཇུག་ཁང་ནས། PAGE -PAGE 2-
